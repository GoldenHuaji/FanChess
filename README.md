# Q & A
> Q: 我该如何运行它？
> 
> A: 
> + Windows: 安装 JRE 环境，然后双击文件即可。
> + Linux: Linux 用户总有办法解决问题的 :)

> Q: 如何走棋？
> 
> A: 红方黑方交替走棋，第一个翻棋子的人是红方。吃子有大小顺序：帅 -> 士 -> 象 -> 车 -> 马 -> 兵。__特别地__，兵 -> 帅，炮 -> 全部（当炮和被吃的子之间有一个棋子时，炮可以吃掉这个棋子，无论该棋子是否翻开，任何**非兵**棋子都可以吃掉炮）
